# DeltaDuckDistributed

## Setup

Start docker container based on node, run like this:

docker run -p 8080:8080 -it -v testvolume:/app vueblack /bin/sh
npm install -g @vue/cli

vue create my-app

Use npm

cd my-app
vue add element

## Vue.js Development
